1. 掌握 HTML 文档的基本结构:
    1.1 创建第一个 HTML 文件
        1.1.1 开发软件: Dreamweaver,Expression Web;
        1.1.2 文件后缀名: .htm 或者 .html;
    1.2 HTML 文档结构
        主要标记: <html>,<head>,<title>,<body>;
        HTML 标记不区分大小写;
        1.2.1 <html>标记
            所有 HTML 文件都是以<html>标记开头,以</html>标记结束;
        1.2.2 <head>标记
            <head>标记是 HTML 文件的头标记,作用是放置 HTML 文件的信息;
            如定义 CSS 样式代码可放置在<head>与</head>标记中;
        1.2.3 <title>标记
            <title>标记为标题标记,可将网页的标题定义在<title>与</title>标记中;
        1.2.4 <body>标记
            <body>是 HTML 页面的主体标记,页面中的所有内容都定义在<body>标记中;
            同样以<body>标记开头,以</body>标记结束;
    1.3 HTML 常用标记
        1.3.1 换行标记
            HTML 语言中换行标记为<br>;
            [例1.3.1]
            <html>
                <head>
                    <title>使用换行标记</title>
                </head>
                <body>
                    <b>
                        黄鹤楼
                    </b><br>
                        故人西辞黄鹤楼,<br>
                        烟花三月......
                </body>
            </html>
        1.3.2 段落标记
            段落标记以<p>标记开头,以</p>标记结束;
            段落标记在段前和段后各添加一个空行,而定义在段落标记中的内容,不受该标记的影响;
        1.3.3 标题标记
            在 HTML 标记中,设定了6个标题标记,分别为<h1>至<h6>;
            其中<h1>代表1级标题...<h6>代表6级标题,数字越小表示级别越高,文字的字体也就越大;
        1.3.4 居中标记
            HTML 页面中的内容默认布局方式是从左到右依次排序;
            <center></center>标记可以使页面中的内容在居中位置显示;
        1.3.5 文字列表标记
            可以将文字以列表的形式依次排序,HTML 中的列表标记主要有无序列表和有序列表两种;
            1.3.5.1 无序列表
                无序列表是在每个列表项的前面添加一个圆点符号;
                通过符号<ul></ul>可以创建一组无序列表,其中每一个列表项以<li>表示;
                [例1.3.5.1]
                ...
                    <ul>
                        <li>Java
                        <li>VB
                        <li>.net
                    </ul>
                ...
            1.3.5.2 有序列表
                使用有序列表标记可以将列表项进行排号,标记为<ol></ol>,每一个列表项前使用<li>;
                [例1.3.5.2]
                ...
                    <ol>
                        <li>Java
                        <li>VB
                        <li>.net
                    </ol>
                ...
    1.4 表格标记
        表格包含标题,表头,行和单元格;
        1.4.1 表格标记<table>
            <table>...</table>标记表示整个表格;<table>标记中有很多属性;
            width 属性设置表格的宽度;
            border 属性设置表格的边框;
            align 属性设置表格的对齐方式;
            bgcolor 属性设置表格的背景色等...
        1.4.2 标题标记<caption>
            <caption></caption>,相关属性: align,valign等;
        1.4.3 表头标记<th>
            <th></th>,也可以通过 align,background,colspan,valign 等属性来设置表头;
        1.4.4 表格行标记<tr>
            <tr></tr>,一组<tr>标记表示表格中的一行;
            <tr>标记需要嵌套在<table>标记中使用,也具有 align,background 等属性;
        1.4.5 单元格标记<td>
            <td>又被称为列标记,一个<tr>标记中可以嵌套若干个<td>标记;
            也具有 align,background,valign 等属性;
    [例1.4]
    ...
    <body>
    <table width="318" height="167" border="1" align="center">
        <caption>学生考试成绩单</caption>
        <tr>
            <td align="center" valign="middle">姓名</td>
            <td align="center" valign="middle">语文</td>
            <td align="center" valign="middle">数学</td>
        </tr>
        <tr>
            <td align="center" valign="middle">张三</td>
            <td align="center" valign="middle">99</td>
            <td align="center" valign="middle">66</td>
        </tr>
        <tr>
            <td align="center" valign="middle">李四</td>
            <td align="center" valign="middle">66</td>
            <td align="center" valign="middle">99</td>
        </tr>
    </table>
    </body>
    ...
    1.5 HTML 表单标记
        1.5.1 <form>...</form>表单标记
            在表单标记中可以定义处理表单数据程序的 URL 地址等信息;
            1.5.1.1 基本语法如下:
                <form action="url" method="get"|"post" name="name" onSubmit="" target="">
                </form>
            1.5.1.2 <form>标记的个属性说明:
                1.5.1.2.1 action 属性
                    用来指定处理表单数据程序的 URL 地址;
                1.5.1.2.2 method 属性
                    用来指定数据传送到服务器的方式;
                    它有两种属性值: get 与 post;
                    get: 将输入的数据追加在 action 指定的地址后面,并传送到服务器;
                    post: 将输入的数据按照 HTTP 协议中的 post 传输方式传送到服务器;
                1.5.1.2.3 name 属性
                    该属性指定表单的名称,其值可以自定义;
                1.5.1.2.4 onSubmit 属性
                    用于指定当用户单击递交按钮时触发的事件;
                1.5.1.2.5 target 属性
                    指定输入数据结果显示在哪个窗口中,其属性值可以设置为: _blank,_self,_parent 和 _top;
                    _blank: 在新窗口中打开目标文件;
                    _self: 在同一个窗口中打开,该项一般不用设置;
                    _parent: 表示在上一级窗口中打开,一般使用框架页时经常使用;
                    _top: 表示在浏览器的整个窗口中打开,忽略任何框架;
            [例1.5.1.2]
            //该例子为创建表单,设置表单名称为 form;当用户递交表单时,递交至 action.html 页面进行处理;
            <form id="form1" name="form" method="post" action="action.html" target="_blank">
            </form>
        1.5.2 <input>表单输入标记
            通过该标记可以向页面中添加单行文本,多行文本,按钮等,语法格式如下:
                <input type="image" disabled="disabled" checked="checked"
                width="digit" height="digit" maxlength="digit" readonly=""
                size="digit" src="un" usemap="un" alt="" name="checkbox" value="checkbox">
        1.5.3 <select>...</select>下拉列表框标记
            此时创建的下拉列表框是一个空的列表,要是用<option>标记向列表中添加内容,格式如下:
                <select name="name" size="digit" multiple="multiple" disabled="disabled">
                </select>
        1.5.4 <textarea>多行文本标记
            与单行文本相比,多行文本可以输入更多的内容;
            通常情况下,<textarea>标记出现在<form>标记的标记内容中,格式如下:
                <textarea cols="digit" rows="digit" name="name" disabled="disabled" readonly="readonly" wrap="value">默认值
                </textarea>
    1.6 超链接与图片标记
        1.6.1 超链接标记
            通过超链接来实现从网站中从一个页面跳转到另一个页面,格式如下:
                <a href=""></a>
                属性 href 用来设定连接到哪个页面中;
        1.6.2 图片标记
            在页面中添加的图片是通过<img>标记来实现的,格式如下:
                <img src="un" width="value" height="value" border="value" alt="提示文字">
