# 第 1 周 android开发环境搭建

## 搭建android开发环境

1. 安装 android studio 工具
2. 选择 SDK(SDK内安装两个版本,开发过程中一个作为最高支持版本,另一个作为最低支持),我选择android P为开发版本,android 7为最低支持版本
3. 选择 AVD(创建模拟器,模拟器内 android 内核选择为当前 SDK 开发版本(P))

## 创建第一个 android 项目

1. 选择创建
2. 选择空项目(empty activity)
3. 填写项目名(myapp1)和包名(com.wxz.myapp1)
4. 选择最低支持android版本(android 7)
