# Visual Studio Code 插件推荐-VSC Netease Music

## Windows

- 在win10上直接在vscode插件商店里面下载即可

## Ubuntu

- 先在vscode插件商店中下载好 VSC Netease Music
- 此时还不能使用
- 打开ubuntu控制台（也称Unix Shell）
- 输入下面命令，使用python方法替换一些文件
- curl https://gist.githubusercontent.com/nondanee/f157bbbccecfe29e48d87273cd02e213/raw | python
- 最好是以管理员身份运行 sudo -s 即可进入管理员模式
